# campo-minado
Basicamente o campo minado em python, botão esquerdo do mouse revela um quadrado o direito adiciona uma bandeira, os numeros indicam quantas bombas há ao redor nos 8 quadrados vizinhos, o objetivo é revelar todos os quadrados sem bomba, e todas as bombas terem uma bandeira, é possivel ganhar só clicando no quadrado no extremo canto inferior direito, caso der problema com o diretório, deve-se abrir a pasta no vscode
