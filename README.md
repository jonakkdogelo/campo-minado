# campo-minado
Basicamente o campo minado em python, botão esquerdo do mouse revela um quadrado o direito adiciona uma bandeira, os numeros indicam quantas bombas há ao redor nos 8 quadrados vizinhos, o objetivo é revelar todos os quadrados sem bomba, as bandeiras são só uma referencia visual, é possivel ganhar só clicando no quadrado no extremo canto inferior direito
